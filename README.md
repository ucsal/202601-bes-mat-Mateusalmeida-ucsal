## Aplicação dos princípios SOLID

### SRP
Separação de responsabilidades nas classes:
- Resposta: controle interno de estado
- Tentativa: encapsulamento da lista
- Questao: separação de validação

### OCP
Melhoria na estrutura para permitir extensão sem modificação direta

### LSP
As classes mantêm comportamento esperado

### ISP
Não aplicável diretamente devido à simplicidade do sistema

### DIP
Redução de dependências diretas através de melhor encapsulamento
